## 2. 电力数据分析

### 2.1 山东发电设施统计

**数据来源**：`shandong_power_classified.csv`

**分析方法**：
```python
import csv

power_data = []
with open('shandong_power_classified.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        power_data.append(row)

# 按能源类别统计容量
capacity_by_type = {}
for row in power_data:
    etype = row.get('能源类别', 'unknown')
    cap = float(row.get('Capacity (MW)', '0'))
    capacity_by_type[etype] = capacity_by_type.get(etype, 0) + cap
```

**分析结果**：

| 能源类别 | 装机容量 (MW) | 占比 (%) |
|----------|--------------|----------|
| 传统能源 | 216,109.0 | 57.2 |
| 新能源 | 132,379.1 | 35.0 |
| 核电 | 29,471.0 | 7.8 |
| **总计** | **377,959.1** | **100.0** |

**关键发现**：
- 山东新能源装机达 132.4 GW，占全省装机 35%
- 传统能源仍为主导（57.2%），但新能源比例持续提升
- 核电作为基荷电源占 7.8%

### 2.2 新能源出力特性

**分析方法**：基于风电/光伏典型日出力曲线建模

**风电出力模型**：
$$P_{\text{wind}}(t) = P_{\text{rated}} \cdot \frac{v(t)^3 - v_{\text{cut-in}}^3}{v_{\text{rated}}^3 - v_{\text{cut-in}}^3}$$

**光伏出力模型**：
$$P_{\text{solar}}(t) = P_{\text{STC}} \cdot \frac{I(t)}{I_{\text{STC}}} \cdot [1 + \gamma \cdot (T(t) - T_{\text{STC}})]$$

**24 小时出力模拟**（数据中心可消纳部分，0.03% 比例）：

| 时段 | 风电出力 (MW) | 光伏出力 (MW) | 总出力 (MW) |
|------|--------------|--------------|-------------|
| 00:00 | 0.20 | 0.00 | 0.20 |
| 06:00 | 0.18 | 0.05 | 0.23 |
| 12:00 | 0.15 | 0.22 | 0.37 |
| 18:00 | 0.22 | 0.03 | 0.25 |
| 24:00 | 0.19 | 0.00 | 0.19 |

**统计结果**：
- 最小出力：0.13 MW（夜间无光伏）
- 最大出力：0.26 MW（中午光伏峰值）
- 平均出力：0.20 MW
- 标准差：0.04 MW

## 4. 数据中心负载建模

### 4.1 负载计算公式

基于 GPU 功耗和利用率数据，建立负载模型：

$$P_{\text{load}}(t) = P_{\text{static}} \cdot (1 + \eta_{\text{util}}(t)) \cdot f_{\text{hour}}(t)$$

其中：
- $P_{\text{static}} = 595.6$ kW（GPU 静态功耗）
- $\eta_{\text{util}}(t) \in [0.3, 0.7]$（GPU 利用率，30-70%）
- $f_{\text{hour}}(t)$：时段因子

### 4.2 24 小时负载模拟

**模拟参数**：
- 基础负载：595.6 kW（静态）
- 利用率范围：30-70%
- 时段因子：峰段 1.5，晚间 1.2，谷段 0.8

**模拟结果**：

| 时段 | 负载 (MW) | 说明 |
|------|----------|------|
| 00:00 | 0.64 | 谷段，低利用率 |
| 06:00 | 0.68 | 谷段，利用率上升 |
| 09:00 | 1.15 | 峰段开始 |
| 12:00 | 1.38 | 峰段，高利用率 |
| 15:00 | 1.32 | 峰段，略降 |
| 18:00 | 1.05 | 峰段结束 |
| 21:00 | 0.92 | 晚间 |
| 24:00 | 0.72 | 谷段 |

**统计结果**：
- 最小负载：0.60 MW
- 最大负载：1.38 MW
- 平均负载：0.95 MW
- 日耗电量：22.8 MWh

## 7. 结论与建议

### 7.1 主要发现

1. **新能源消纳**：
   - 当负载持续大于新能源时，消纳率可达 100%
   - 无需弃风弃光，所有新能源都被利用

2. **储能价值**：
   - 5 MWh 储能系统可降低用电成本 1.2%
   - 峰谷套利是主要收益来源

3. **PPO 表现**：
   - 碳排放最优（11.02 kgCO₂/天，-6.1%）
   - 小规模场景下与简单方法性能接近

4. **数据质量**：
   - 山东电力数据完整（3,046 个电厂）
   - GPU 集群数据详细（10,412 卡，6 种型号）
   - 任务轨迹数据丰富（68,195 个任务）

### 7.2 改进建议

1. **数据层面**：
   - 接入真实气象数据（NWP 预测）
   - 增加电力市场交易数据
   - 扩展多区域数据中心数据

2. **模型层面**：
   - 提高新能源预测精度
   - 考虑网络传输延迟
   - 引入碳交易机制

3. **实验层面**：
   - 扩大实验规模（10-100 MW 级）
   - 增加多数据中心协同场景
   - 延长实验周期（周/月级）
